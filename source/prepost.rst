本书主要内容
============

本书的内容组织首先从 :ref:`C 语言入门` 开始, 在此卷中, 讲解 C 语言的语法,
编译程序的步骤, 以及对可执行程序和内存结构的简单科普. 并且学习 :ref:`C 语言标准库` 的使用.

然后, 进入两个分支, 一个分支是进入 C++ 的学习, 从 :ref:`从 C 扩展到 C++` 开始, 然后按照:

1. :ref:`在 C++ 中摆脱 C`
#. :ref:`C++ 标准库与标准模板库`
#. :ref:`C++ 新特性`
#. :ref:`C++ 编程模式`

的顺序学习 C++ 语言.

或者通过 C 语言了解计算机、 操作系统以及简单的数据结构与算法.

1. :ref:`C 编程模式`
#. :ref:`通过 C 语言认识计算机`
#. :ref:`C 数据结构与算法`
