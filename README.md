5. C/C++ 的标准库使用参考
6. 开发中常用的工具以及包管理方法
7. Linux 系统下的 C/C++ 编程
8. 通过 C 语言认识计算机的组成
9. 简单数据结构以及算法的 C 语言实现
10. C/C++ 内联汇编
11. C++ 实现 Linux 网络编程: socket, TCP, UDP, HTTP
