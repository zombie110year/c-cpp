7. Linux 系统下的 C/C++ 编程

9.  简单数据结构以及算法的 C 语言实现
10. C/C++ 内联汇编
11. C/C++ 实现 Linux 网络编程: socket, TCP, UDP, HTTP

- 开发中常用的工具以及包管理方法
