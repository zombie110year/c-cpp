#####################
C++ 从 C 继承的标准库
#####################

C++ 将 C 的标准库进行了封装, 内部所有对象都被包装进了 ``std`` 命名空间中.

================= ==================
C 的标准库头文件  C++ 的标准库头文件
----------------- ------------------
assert.h          cassert
ctype.h           cctype
errno.h           cerrno
float.h           cfloat
limits.h          climits
locale.h          clocale
math.h            cmath
setjmp.h          csetjmp
signal.h          csignal
stdarg.h          cstdarg
stddef.h          cstddef
stdio.h           cstdio
stdlib.h          cstdlib
string.h          cstring
time.h            ctime
================= ==================
